## Project Overview

Retail businesses generate massive amounts of transactional data every day. Extracting meaningful insights from this data is essential for understanding customer behavior, optimizing sales strategies, and making informed business decisions.

In this project, I performed a comprehensive Exploratory Data Analysis (EDA) on a retail sales dataset to identify trends, customer demographics, purchasing behavior, and revenue patterns using Python and its data analysis ecosystem.

---

## Objectives

The analysis aims to answer key business questions such as:

* Which product categories generate the highest sales?
* How do customer demographics influence purchasing behavior?
* Are there seasonal or monthly sales trends?
* What relationships exist between numerical variables?
* What actionable insights can help improve business performance?

The dataset contains retail transaction records, including customer demographics, purchase details, product categories, quantities purchased, pricing information, and transaction dates.

# Key Insights

Some notable findings from the analysis include:

* Electronics generated the highest overall revenue.
* Clothing recorded a high purchase frequency among customers.
* Sales exhibited noticeable monthly and quarterly variations.
* Female customers represented a slightly larger proportion of the customer base.
* Numerical variables showed relatively weak correlations, suggesting that purchasing behavior depends on multiple factors beyond age or quantity alone.

---

# Business Recommendations

Based on the analysis:

* Prioritize inventory for high-performing product categories.
* Increase marketing efforts during peak sales periods.
* Develop targeted campaigns for different customer demographics.
* Monitor seasonal purchasing patterns to optimize stock levels.
* Use customer purchasing behavior to inform promotional strategies.

# How to Run

```bash

```

3. Install dependencies


4. Launch Jupyter Notebook


5. Open the notebook and run all cells.
